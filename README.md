# Event Clustering Benchmark

## Идея проекта

Этот проект демонстрирует **кластеризацию событий в реальном времени**.  
В реальной инфраструктуре сотни и тысячи событий могут поступать от разных систем (например, Cloud, Network, Hardware, Software, SupportPortal). Эти события нужно группировать в кластеры, чтобы специалисты могли оперативно реагировать на инциденты, не теряя важную информацию.  

Проект позволяет:

- Генерировать синтетические события с разными типами данных (`title`, `description`, `category`, `severity`, `class`)  
- Моделировать поток событий с временными метками  
- Кластеризовать события по системе и по содержимому события разными методами:  
  - Hashing + Cosine Similarity  
  - TF-IDF + Cosine Similarity  
  - Jaccard Similarity  
  - FAISS (Approximate Nearest Neighbor)  
- Собирать метрики производительности, включая **время обработки**, **количество кластеров** и распределение событий по кластерам  
- Сохранять результаты в CSV и TXT для анализа  
- Визуализировать кластеры с помощью PCA (опционально)
